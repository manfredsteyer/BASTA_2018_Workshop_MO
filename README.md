# Angular Demo: BASTA 2018 Workshop

## Prerequisites

- NodeJS
- Angular CLI: npm i -g @angular/CLI

## Start
```
ng serve
```

## Contents:

This example shows:

- Components
- Data Binding
- HTTP Access
- Services and Dependency Injection
- Treeshakable Providers
- Pipes
- Modules
- Routing
- Lazy Loading and Preloading
